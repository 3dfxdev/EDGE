
EUREKA DOOM EDITOR

Features:

-  can load/view/edit/save maps in a wad file

-  create new maps from scratch [planned]

-  Undo/Redo capability [in progress]

-  3D preview

-  native support for EDGE features (RTS) [planned]


Requirements:

-  128 MB of computer memory

-  800x600 or higher screen resolution

-  the data (iwad) file from DOOM, DOOM II or FreeDoom


Compiling Requirements:

-  GNU make

-  C++ compiler

-  the FLTK GUI toolkit (v1.1.9 or later)
