
Edge Libraries
==============

Internal (in SVN)
-----------------

EPI      : Edge Platform Interface (utility code)
Deh_Edge : DeHackEd/BEX patch to DDF converter
LZO      : Compression library


Affiliated (in snapshots, not in SVN)
-------------------------------------

glBSP    : GL-Nodes builder
Humidity : MIDI playback library


External
--------

LibSDL  : Simple DirectMedia Library (video and input)

OpenGL  : 3D Graphics library
OpenAL  : 3D Sound library

OGG + Vorbis + VorbisFile : Compressed sound and music

FLTK    : Fast Light ToolKit, for error dialogs [Linux only]

HawkNL  : Networking library [NOTE: not currently used]

LibPNG  : PNG image handling (Portable Network Graphics)
Jpeglib : JPEG image handling
Zlib    : Compression library (used by LibPNG)
